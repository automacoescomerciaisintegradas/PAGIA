# Especificação: Setup do Sistema

## Objetivo
Configurar a base sólida para o desenvolvimento do sistema Backend + Frontend.

## Requisitos
- Estrutura de diretórios organizada.
- Configuração de TypeScript e scripts de build.
- Base para o servidor Express.
- Template base para o Frontend.

## Critérios de Aceitação
- Servidor roda localmente.
- Frontend servido pelo backend ou acessível via dev server.
- Conexão base com agentes PAGIA funcional.
