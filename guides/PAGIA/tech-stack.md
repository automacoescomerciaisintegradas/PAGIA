# Tech Stack do Projeto

## Backend
- **Core**: Node.js (v18+)
- **Linguagem**: TypeScript
- **API**: Express.js
- **Banco de Dados**: Supabase (PostgreSQL) / Redis para cache
- **IA**: Gemini 2.0 Flash, Groq (Llama 3.3)

## Frontend
- **Framework**: HTML5, Vanilla JavaScript (ou Next.js conforme solicitado)
- **Styling**: Vanilla CSS / Flexbox / Grid
- **Animações**: CSS Animations / Framer Motion (se React)

## DevOps & Ferramentas
- **Versionamento**: Git
- **Ambiente**: Docker
- **Workflow**: PAGIA Conductor
